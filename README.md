# Бюджетный помощник

## Описание проекта

«Бюджетный помощник» — это простая консольная программа на Python.
Она хранит ежедневные расходы пользователя за один месяц. Пользователь
может добавить расход, посмотреть все записи и быстро посчитать сумму
расходов за период с дня A по день B.

Проект выполнен в учебном стиле для дисциплины «Основы программирования».
В программе специально показаны структуры данных и алгоритмы, изученные
в курсе. Базы данных, графический интерфейс и внешние библиотеки не
используются.

## Функционал

- добавление расхода: день, сумма, категория;
- просмотр всех расходов;
- подсчет суммы расходов за период;
- поиск дня с максимальными расходами;
- сортировка категорий по сумме трат;
- отмена последнего добавленного расхода;
- вывод дерева расходов по дням.

## Используемые структуры данных и алгоритмы

### Префиксные суммы

Расходы по дням хранятся в списке:

```python
daily_expenses = [0] * 32
```

Индекс 0 не используется, поэтому день месяца можно использовать как
индекс списка.

Для быстрого ответа на запрос суммы расходов за период используется
массив префиксных сумм:

```python
prefix_sums = [0] * 32
```

`prefix_sums[i]` хранит сумму расходов с 1 по i день включительно.
После добавления или отмены расхода массив пересчитывается.

Сумма расходов с дня A по день B считается по формуле:

```python
prefix_sums[B] - prefix_sums[A - 1]
```

Сложность самого запроса: **O(1)**, потому что массив префиксных сумм
уже построен заранее.

### Линейный поиск

Линейный поиск используется для поиска дня с максимальной суммой расходов.
Программа проходит по списку `daily_expenses` с 1 по 31 день и сравнивает
значения.

Сложность: **O(n)**, где `n = 31`.

### Сортировка вставками

Сортировка вставками используется для сортировки категорий по сумме трат
по убыванию. Самые дорогие категории выводятся первыми.

В проекте не используются `sort()` и `sorted()`, потому что по заданию
нужно реализовать сортировку вручную.

Сложность: **O(k²)**, где `k` — количество категорий.

### Стек

Стек используется для отмены последнего добавленного расхода.
Каждый новый расход добавляется в список `undo_stack`. При отмене
программа достает последний расход с помощью `pop()`.

После отмены:

- расход удаляется из общего списка расходов;
- сумма расходов за день уменьшается;
- массив префиксных сумм пересчитывается;
- дерево расходов перестраивается.

### Бинарное дерево поиска

Бинарное дерево поиска используется для хранения расходов по дням.
Ключ узла дерева — день месяца. В каждом узле хранится список расходов
за этот день.

При выводе дерева используется рекурсивный симметричный обход:

```text
левое поддерево -> текущий узел -> правое поддерево
```

Поэтому расходы выводятся по возрастанию дней.

## Как запустить

Перейдите в папку проекта:

```bash
cd budget_helper
```

Запустите программу:

```bash
python main.py
```

или:

```bash
python3 main.py
```

При запуске программа спросит, нужно ли загрузить демонстрационные
данные. Эти данные нужны только для удобной проверки работы программы.

Демонстрационные расходы:

- 1 день — 500 руб. — еда;
- 2 день — 120 руб. — транспорт;
- 5 день — 1500 руб. — одежда;
- 5 день — 700 руб. — еда;
- 10 день — 3000 руб. — техника;
- 15 день — 900 руб. — кафе;
- 20 день — 200 руб. — транспорт;
- 25 день — 2500 руб. — подарки.
